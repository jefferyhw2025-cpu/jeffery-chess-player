Jeffery Chess Privacy Policy

Last updated: June 16, 2026

Jeffery Chess is a chess game for learning and playing chess. The iOS app does not require an account and does not collect personal information.

Data Collection

The iOS app stores game progress, settings, local accounts, and feedback locally on your device. It does not send this information to the developer.

Hosted Web Feedback

If you use the feedback form on the hosted web version, the feedback message and related game context may be submitted to the developer through Netlify Forms. This feedback is used only to improve the game.

Hosted Web Ranked Matches

If you play a ranked match on the hosted web version, Jeffery Chess may send the PGN move record, final result, AI level, player ID, and rank metadata to the server. This is used only to verify ranked results and store official rank progress. It is not used for advertising tracking.

Hosted Web Professional League

If you play Professional League mode on the hosted web version, Jeffery Chess may store your player ID, display name, league points, wins, draws, losses, season ID, and update time so the leaderboard can show account-bound standings.

Hosted Web Cloud Backup

If you use cloud backup on the hosted web version, Jeffery Chess may store a safe player profile backup through Netlify. This backup is limited to whitelisted game progress such as rank points, achievements, daily tasks, settings, and saved game state. It does not include password hashes, feedback text, administrator data, API keys, or private temporary IDs.

Local Game Data

The app may store local game preferences or score information on your device so the game can remember your settings. This information stays on your device and is not sent to us.

Local Account Deletion

If you create a local account in the iOS app, you can delete it inside the app from Player Profile or Settings. Deleting a local account clears that account's rank, achievements, daily tasks, league progress, and login data from the device.

Tracking

We do not sell personal data, run advertising tracking, or use third-party analytics in Jeffery Chess.

Children's Privacy

The app is designed as a general chess game and does not knowingly collect information from children.

Contact

For privacy questions about Jeffery Chess, contact the app developer through the support information listed on the App Store page.
