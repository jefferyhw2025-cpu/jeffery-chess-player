MateQuest Chess Privacy Policy

Last updated: June 25, 2026

MateQuest Chess is a chess game for learning and playing chess. The iOS app does not require an account for local play. Most progress is stored on your device. If you choose optional online features, limited game data may be transmitted only to provide that feature.

Data Collection

The iOS app stores game progress, settings, local accounts, and feedback locally on your device. This local data is not sent to the developer unless you choose an online feature such as feedback, cloud backup, direct play signaling, or Game Center.

Hosted Web Feedback

If you use the feedback form on a hosted version, the feedback message and related safe diagnostics may be submitted to the developer through Netlify Forms. Diagnostics can include app version, device/browser type, current language, network mode, room state, Game Center state, current page, FEN, and PGN. This feedback is used only to improve the game.

Hosted Web Ranked Matches

If you play a ranked match on the hosted web version, MateQuest Chess may send the PGN move record, final result, AI level, player ID, and rank metadata to the server. This is used only to verify ranked results and store official rank progress. It is not used for advertising tracking.

Hosted Web Professional League

If you play Professional League mode on the hosted web version, Jeffery Chess may store your player ID, display name, league points, wins, draws, losses, season ID, and update time so the leaderboard can show account-bound standings.

Hosted Web Cloud Backup

If you use cloud backup on the hosted web version, MateQuest Chess may store a safe player profile backup through Netlify. This backup is limited to whitelisted game progress such as rank points, achievements, daily tasks, settings, and saved game state. It does not include password hashes, feedback text, administrator data, API keys, or private temporary IDs.

Direct Play Signaling

If you use automatic room-code exchange for WebRTC direct play, the game may send a temporary room code, WebRTC invite/answer signaling text, and app version to Netlify so two players can connect. This signaling data is used only to start the direct-play match, expires after about 30 minutes, and is not used for advertising tracking.

Game Center

If you choose Game Center features in the iOS app, Apple may process your Game Center player identity, matchmaking, leaderboard, achievement, and multiplayer information. MateQuest Chess uses Game Center only for gameplay features such as sign-in, matchmaking, achievements, and leaderboards. It is not used for advertising tracking.

Local Game Data

The app may store local game preferences or score information on your device so the game can remember your settings. This information stays on your device and is not sent to us.

Local Account Deletion

If you create a local account in the iOS app, you can delete it inside the app from Player Profile or Settings. Deleting a local account clears that account's rank, achievements, daily tasks, league progress, and login data from the device.

Tracking

We do not sell personal data, run advertising tracking, or use third-party analytics in MateQuest Chess.

Children's Privacy

The app is designed as a general chess game and does not knowingly collect information from children.

Contact

For privacy questions about MateQuest Chess, contact the app developer through the support information listed on the App Store page.
